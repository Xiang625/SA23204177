Overview

SA23204177 is a simple R package developed to test whether the sample mean vector is a zero vector (Wang and Shao, 2020) using R, Rcpp and RcppEigen (implemented through the R package Rcpp and RcppEigen). Both R and Rcpp versions are produced. Namely Wang2020 for R and Wang2020C for C++.

The R package ‘microbenchmark’ can be used to benchmark the above R and C++ functions.

Details

Suppose we have a \(p\)-dimensional stationary nonlinear time series \(Y_t=\mu+g(\epsilon_t,\epsilon_{t-1},\cdots)\), for some measurable function \(g\), where \(\{\epsilon_t\}_{t=-\infty}^\infty\) are i.i.d. random elements in some measurable space.

For \(r \in [0,1]\), define \[ T_n(r):=\sum_{t=1}^{\lfloor nr\rfloor}\sum_{s=1}^tY_{t+m}^TY_s. \] The test statistic is defined as: \[ T_{SN,n}:=\frac{T_n(1)^2}{W_n^2}, \] where \(W_n^2=\frac1n\sum_{k=1}^n\left(T_n(k/n)-\frac{k(k+1)}{n(n+1)}T_n(1)\right)^2\).

Source code of Wang2020 and Wang2020C

The source R code for Wang2020 is as follows:

T_n <- function(r,n1,YY,m){
  t_n = 0
  n = floor(r*n1)
  for (t in 1:n){
    t_n = t_n + sum(YY[t+m,1:t])
  }
  return(t_n)
}

Wang2020 <- function(sam,m,n){
  n1 = n-m
  YY = sam%*%t(sam)
  W <- NULL
  for (k in 1:n1){
    W[k] = (T_n(k/n1,n1,YY,m)-k*(k+1)/n1/(n1+1)*T_n(1,n1,YY,m))^2
  }
  W_n = mean(W)
  test = T_n(1,n1,YY,m)^2/W_n
  reject = test > 54.70
  return(reject)
}

The source C++ code for Wang2020C is as follows:

#include <RcppEigen.h>
using namespace Rcpp;
// [[Rcpp::depends(RcppEigen)]]
//[[Rcpp::export]]
double T_nC(double r, int n1, const Eigen::MatrixXd& YY, int m){
  double t_n = 0;
  int n = r*n1;
  for (int i=1; i<=n; i++){
    for (int j=1; j<=i; j++){
      t_n = t_n + YY(i+m-1,j-1);
    }
  }
  return t_n;
}

//[[Rcpp::export]]
bool Wang2020C(const Eigen::MatrixXd& sam, int m, int n){
  double n1 = n - m;
  Eigen::MatrixXd YY = sam * sam.adjoint();
  double W = 0;
  for (int k=1; k <= n1; k++){
    W += pow(T_nC(k/n1,n1,YY,m)-k*(k+1)/n1/(n1+1)*T_nC(1,n1,YY,m),2);
  }
  double W_n = W / n1;
  double test = pow(T_nC(1,n1,YY,m),2)/W_n;
  bool reject = test > 54.70;
  return reject;
}
